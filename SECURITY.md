# Security Policy

## Supported Versions

The following versions of ha-alplakes are
currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
|   0.4.1 | :white_check_mark: |
|   0.3.0 | :x:                |
| < 0.3.0 | :x:                |

## Reporting a Vulnerability

Please report any vulnerability using Github Issues.

Updates are provided on a best-effort basis, depending on severity.
